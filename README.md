# OCR2WhatsApp
Python project that get numbers from images then send messages through whatsapp <br><br>
## How to install:
- 1: pip install -r requirements.txt
- 2: go to https://github.com/UB-Mannheim/tesseract/wiki and install windows version
- 3: add Tesseract OCR to path (example: C:\Program Files\Tesseract-OCR)
